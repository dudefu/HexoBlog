## 法一
```Java
//求出最大年龄,求出最小年龄,求出平均年龄
        int[] ages = {34,11,45,23,16};
        boolean flag = true ;
        for (int i = 4; flag && i >0 ; i--) {
            flag = false ;
            for (int j = 0; j < i; j++) {
                if(ages[j]>ages[j+1]){
                    int temp = ages[j];
                    ages[j] = ages[j+1];
                    ages[j+1] = temp ;
                    flag = true ;
                }
            }
            System.out.println("-------");
        }
        for (int i = 0; i < 5; i++) {
            System.out.print(ages[i]+ "\t");
        }
```
